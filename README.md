📊 Student Data Analysis Project
📌 Project Overview

This project focuses on analyzing sales data using Excel.
The goal was to understand sales performance across countries, products, and channels.
I used Pivot Tables, charts, and slicers to explore the data and build an interactive dashboard.
The analysis helped identify top-performing countries, high-profit products, and the most effective sales channels.

🔍 Key Insights

Certain product categories contribute more to overall sales 
Sales vary based on payment type, showing its impact on customer behavior
